<h1 align="center">🍽️ Recipe Finder App</h1>

<p align="center">
A simple yet powerful Recipe Finder application built using <b>HTML</b>, <b>CSS</b>, and <b>JavaScript</b>. The app uses the <a href="https://www.themealdb.com/api.php" target="_blank">TheMealDB API</a> to search and display delicious recipes with ingredients and cooking instructions.
</p>

<h2>🚀 Features</h2>

<ul>
<li>🔍 Search recipes by dish name.</li>
<li>🍲 View recipe details including ingredients and instructions.</li>
<li>🌎 Displays recipe origin/cuisine.</li>
<li>🖼️ Shows recipe image.</li>
<li>📱 Fully responsive design.</li>
<li>✨ Clean and modern user interface.</li>
<li>⚡ Fast API-based recipe search.</li>
</ul>

<hr>

<h2>⚙️ How It Works</h2>

<ol>
<li>User enters the name of a dish.</li>
<li>Click the <b>Search</b> button.</li>
<li>The application sends a request to the <b>TheMealDB API</b>.</li>
<li>The API returns recipe information.</li>
<li>The app displays recipe image, ingredients, origin, and instructions.</li>
<li>Click <b>View Recipe</b> to read the complete cooking steps.</li>
</ol>
